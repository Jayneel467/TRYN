# Brand assets

Official TRYN Studios logo files (PNG):

| File | Usage |
|------|--------|
| `tryn-logo-light.png` | Full logo — light backgrounds (navy wordmark, transparent) |
| `tryn-logo-dark.png` | Full logo — dark backgrounds (white wordmark, transparent) |
| `tryn-logo-header-light.png` | Header / compact — light backgrounds (transparent, navy wordmark) |
| `tryn-logo-header-dark.png` | Header / compact — dark backgrounds (transparent, white wordmark) |
| `itinero-logo.svg` | Parent company (Itinero Travels) |

All PNGs have baked backgrounds removed for transparent overlay use (header, footer, hero).

Regenerate transparent PNGs from source files with `python scripts/process-logos.py`.
